# AuraChef Backend

A TypeScript backend application.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Development:
   ```bash
   npm run dev
   ```

3. Build:
   ```bash
   npm run build
   ```

4. Start production:
   ```bash
   npm start
   ```

## Project Structure

- `src/` - Source code
- `dist/` - Compiled JavaScript (generated)
- `tests/` - Test files

## Development

This project uses TypeScript for type safety and includes basic testing setup with Jest.
